# 🏍️ ResQRide
### Motorcycle Roadside Emergency Assistance Mobile Application

ResQRide is a Final Year Project (FYP) developed to provide fast and reliable roadside emergency assistance for motorcyclists in Malaysia. The application connects riders experiencing motorcycle breakdowns with nearby workshops that provide mobile repair services, allowing mechanics to travel directly to the rider's location.

---

# 🎯 Objectives

- Improve emergency response time during motorcycle breakdowns.
- Enhance rider safety through real-time assistance.
- Help riders locate nearby workshops quickly.
- Strengthen collaboration between riders and local workshops.
- Provide a complete motorcycle emergency assistance platform.

---

# 🔒 Firebase Configuration

This repository **does not include** the following file:

```
app/google-services.json
```

To run this project:

1. Create your own Firebase project.
2. Download `google-services.json`.
3. Place it inside:

```
app/
```

---

# 🔑 Google Maps API

Replace the API key in the project with your own Google Maps API Key.

Required APIs:

- Maps SDK for Android
- Places API
- Directions API
